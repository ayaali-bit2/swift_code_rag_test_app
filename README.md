# swift_code_rag_test_app

A lightweight Swift executable showcasing structured logic and basic unit coverage as a Swift package.

## Overview

- **Executable**: `SimpleSwiftApp` under `Sources/SimpleSwiftApp`
- **Logic Highlights**:
  - Greeting generation tailored to a name
  - Featured quote selection
  - Feature highlights list
  - Sequence summary builder
- **Tests**: Located in `Tests/SimpleSwiftAppTests`, covering greetings, featured quote formatting, feature highlights, and sequence summary output.

## Requirements

- Swift 5.9+
- macOS 14+ (as specified in `Package.swift` platforms)

## Getting Started

```bash
swift build
swift test
swift run SimpleSwiftApp
